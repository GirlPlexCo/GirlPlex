Who We Are

Interested in Computer Science and coding?

GirlPlex is coding summer camp for girls from grades 6 through 8. Over the course of two weeks, girls learn how to create their own websites, using languages such as Ruby, HTML, CSS, and JavaScript.

Apply

Program

Closing the gender gap, one girl at a time.

GirlPlex is focused on closing the gender gap in computer science. Currently, women are very underrepresented in computer science fields. GirlPlex aims to educated girls in computer science from middle school, so they will be more likely to pursue computer science fields in high school.

Learn More

Javascript

JavaScript is a client-scripting language used in web development. It can be used as either a front-end or back-end language.

HTML and CSS

HTML and CSS are front-end languages used to design web pages. They are essential languages to creating websites.

Ruby

Ruby is object-oriented coding language that can be used as a backend language. It can be used to create dynamic websites.

Statistics of Girls in Technology

There is a projected growth of females in computer science by 15-20% from 2012-2022, however, even with this growth most of the computer science jobs will be held by men. As more STEM careers are added everyday, the clear absence of females is notable.

There is a miscommunication between the computer science industry and the message girls recieve about being successful in tech. GirlPlex strives to change that and help girls become more confident in the technology industry.
